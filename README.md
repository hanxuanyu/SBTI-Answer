## 人格分析原理
- 常规题共 30 道，按 15 个维度计分；隐藏触发题共 2 道。
- 每题选项都会映射到 1/2/3 分，同一维度累计得分后换算为 L/M/H：小于等于 3 为 L，等于 4 为 M，更高则为 H。
- 常规人格会把 15 维模式串中的 L/M/H 映射为 1/2/3，再与答题结果逐维比较。
- 距离值 = 各维差值绝对值之和，精准命中 = 完全一致的维度数量，匹配度 = max(0, round((1 - 距离值 / 30) * 100))。
- 若饮酒隐藏题命中，结果直接覆盖为 DRUNK；若所有常规人格的最高匹配度仍低于 60%，结果兜底为 HHHH。
- 文档中的“标准答案”不是唯一解，而是一组能稳定打到该人格维度模式、并通过同一套匹配逻辑验证的合法答案组合。

## 15个维度列表

- S1 自尊自信（自我模型）
  - L：对自己下手比别人还狠，夸你两句你都想先验明真伪。
  - M：自信值随天气波动，顺风能飞，逆风先缩。
  - H：心里对自己大致有数，不太会被路人一句话打散。

- S2 自我清晰度（自我模型）
  - L：容易被外界带着走，问到‘你到底想要什么’时脑子先起雾。
  - M：偶尔能对上自己的频道，但杂音一多就开始怀疑人生。
  - H：知道自己在乎什么、讨厌什么，做选择时心里不容易跑偏。

- S3 核心价值（自我模型）
  - L：评价体系主要看外部眼色，夸和踩都能把心情拨得很响。
  - M：嘴上说有自己的标准，真遇事还是会偷偷参考大众答案。
  - H：有一套相对稳定的内在坐标，外界热闹归热闹，你不太乱。

- E1 依恋安全感（情感模型）
  - L：消息一慢、语气一淡，脑内小剧场立刻连播八集。
  - M：能努力告诉自己别多想，但心里还是会来回试探。
  - H：关系里基本能信任对方，不会把每次停顿都理解成告别。

- E2 情感投入度（情感模型）
  - L：感情可以有，但不轻易全押，心里总留着一条撤退通道。
  - M：投入看状态和对象，能热起来，也留得住一点清醒。
  - H：一旦认定就很舍得给真心，容易把关系放进生活核心区。

- E3 边界与依赖（情感模型）
  - L：亲近和黏人会让你安心，距离一拉开反而容易不踏实。
  - M：能亲近也能独处，舒服的距离主要看人下菜碟。
  - H：很重视个人空间，再亲密的关系也希望彼此留门不锁死。

- A1 世界观倾向（态度模型）
  - L：看世界先开防骗模式，善意要过安检，糖也像带钩。
  - M：对人性保持观望，不先神化，也不急着判死刑。
  - H：愿意先把人往好处想，默认世界不完美，但也没坏到透。

- A2 规则与灵活度（态度模型）
  - L：规矩能绕就绕，感觉对了先上，制度排在体验后面。
  - M：会衡量规则和现实，既不盲冲，也不死守。
  - H：对秩序和边界比较在意，能按流程走的事不想硬拐弯。

- A3 人生意义感（态度模型）
  - L：人生意义感时有掉线，很多事做着做着就像在走流程。
  - M：偶尔觉得一切有点虚，但还愿意继续找自己的解释。
  - H：做事通常有方向感，心里多少有个‘我为什么在这里’。

- Ac1 动机导向（行动驱力模型）
  - L：更容易被风险和麻烦卡住，推进一件事前先想怎么别翻车。
  - M：想进步也怕受挫，油门和刹车经常一起踩。
  - H：更偏成果导向，既然要做，就想把事情往前拱一大截。

- Ac2 决策风格（行动驱力模型）
  - L：选项一多就开始内耗，决定像拉扯战，迟迟按不下确认键。
  - M：大方向能定，小细节还是会磨一会儿。
  - H：下判断相对利落，不爱在同一个岔路口反复兜圈。

- Ac3 执行模式（行动驱力模型）
  - L：计划经常写得比做得认真，动力多半靠截止日期追着跑。
  - M：能做，但稳定性看状态，发挥像间歇性在线。
  - H：想到的事倾向于尽快推进，既能排步骤，也能盯结果。

- So1 社交主动性（社交模型）
  - L：社交启动比较慢，新场子先观察，热闹一般轮不到你第一个冲。
  - M：不主动也不抗拒，有人来搭话你能接住，没局也能自得其乐。
  - H：见人见场都比较能开机，愿意主动破冰，也不怕把气氛带热。

- So2 人际边界感（社交模型）
  - L：亲近感来得快，熟人一旦对上频率，很容易往自己人里收。
  - M：有边界，但弹性不小，亲疏远近会跟着关系走。
  - H：边界感自带警报器，哪怕关系不错，也希望彼此别越线。

- So3 表达与真实度（社交模型）
  - L：想法和表达基本同频，不太爱演，也懒得切太多版本的自己。
  - M：会看场合调节表达，但大体还算保留原装内核。
  - H：很会按对象切换呈现方式，真实不一定缺席，只是包装经常先上场。
